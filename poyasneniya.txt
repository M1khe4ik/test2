Общие пояснения
main.cpp:
Подключаются необходимые заголовочные файлы.
Создается обработчик сигналов для корректного завершения программы при прерывании.
Парсятся аргументы командной строки для получения порта, максимального количества клиентов и файла дампа.
Настраивается логирование с использованием библиотеки spdlog.
Создается объект сервера, загружаются данные из файла дампа (если указан), и сервер запускается.

CMakeLists.txt:
Устанавливается минимальная версия CMake и стандарт C++.
Добавляются поддиректории для библиотек spdlog и googletest.
Определяются исполняемые файлы и их зависимости.
Включается поддержка тестирования и добавляются тесты.

server.h:
Определение класса Server с методами для запуска сервера, обработки клиентов и команд.
Объявляется объект storage_ как public для доступа из main.cpp.

run.sh и test.sh:
Скрипты для сборки и запуска программы, а также для запуска тестов.

server.cpp:
Создание сокета и привязка его к указанному порту.
Прослушивание подключений и обработка каждого клиента в отдельном потоке.
Обработка команд от клиента (PUT, GET, DEL, COUNT, DUMP) и взаимодействие с хранилищем данных.

storage.cpp:
Реализация методов для работы с хранилищем данных: добавление, получение, удаление значений по ключу, подсчет количества ключей, сохранение данных в файл и загрузка данных из файла.
Использование мьютекса для обеспечения потокобезопасности.

storage.h:
Определение класса Storage и его методов для работы с хранилищем данных.

main_test.cpp:
Тестирование основных методов хранилища с использованием библиотеки Google Test.
